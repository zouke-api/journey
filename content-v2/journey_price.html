行程报价

文档更新日期 2018-06-06：

所有接口中code处理逻辑：

price_type  sub 分项报价 | total 总报价 | day 按天报价 | "" 未开通报价 
code 0 是正常返回
code 大于0 和后台约束处理逻辑
code 小于0 直接提示msg信息

1. 获取报价详情

URL：/journey/pc/get-price-info

描述：仅用于报价编辑

Request：

{
  "jId":36000
}

参数说明：

无

Response：

{
  "code": 0,
  "info": {
    // 总报价
    "total_price": 0,
    // 分项报价
    "sub_item_price": [
      {
        "type": "hotel",
        "name": "客人酒店",
        "price": 10,
        "desc": ""
      },
      {
        "type": "tip",
        "name": "司导小费",
        "price": 0,
        "desc": ""
      }
    ],
    // 按天报价
    "day_price": {
      "981743": [
        {
          "type": "hotel",
          "name": "客人酒店",
          "price": 0,
          "desc": ""
        },
        {
          "type": "fight-ticket",
          "name": "客人机票",
          "price": 0,
          "desc": ""
        }
      ]
    }
  },
  "msg": "成功",
  "timestamp": 1528274310
}

2. 总价的保存

URL：/journey/app/update-total-price

描述：仅用于报价编辑

Request：

{
  "jId":70701,
  "price":1000
}

参数说明：

无

Response：

{
  "code": 0,
  "msg": "成功",
  "timestamp": 1528273060
}
