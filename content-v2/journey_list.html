行程列表

文档更新日期 2018-06-06：

所有接口中code处理逻辑：

code 0 是正常返回
code 大于0 和后台约束处理逻辑
code 小于0 直接提示msg信息

1. 获取行程首页列表

URL：/journey/pc/get-list

描述：行程列表

Request：

{
    "status":"all", 
    "page":0,
    "size":5
}

参数说明：

“系统”后期再加入，先完成“我的 待出发 出行中 已结束”
status：all 我的 | soon 待出发 | start 出行中 | end 已结束
page：从0开始

Response：

{
    "code": 0,
    "list": [
        {
            "id": 70695,
            "title": "澳大利亚双城深入8日行",
            "img": "https://cdn1.zouke.com/journey/edit/1498133060255_5838_NjAuanBn.jpg",
            "totalDays": 8,
            "startDate": "2018-06-06",
            "stat": "start",  // 状态 none 未设置出行时间 | pending 待出发 | start 出行中 | end 已结束
            "visitNum": 0, // 查看
            "touristNum": 0, // 成行
            "modifyTime": "2018-06-06 13:04" // 更新时间
        }
    ],
    "msg": "成功",
    "sum": 1,
    "timestamp": 1528265370
}

2. 列表页中的复制接口

URL：/journey/pc/clone

描述：复制指定行程

Request：

{
    "jId": 3602226
}

参数说明：

jId 需要复制的行程id
