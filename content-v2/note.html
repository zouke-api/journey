1. /tr-media/pc-note/list

描述：笔记相册列表 - 修改05-28

request：

{
    "keywords": "巴黎",
    "cat": "NT",
    "pageNo": 0,
    "pageSize": 10
}

说明：cat：SY系统、MI我的、NT笔记、AB相册

response

{
  "code": 0,
  "list": [
    {
      "id": 10000,
      "title": "发现西西里",
      "updateTime": "2018-05-15 12:12:21",
      "viewTimes": 100,
      "image": "https://cdn1.zouke.com/journey/collection/usa.jpg",
      "type": "NT"
    },
    {
      "id": -10001,
      "title": "发现西西里",
      "updateTime": "2018-05-15 12:12:21",
      "viewTimes": 100,
      "image": "https://cdn1.zouke.com/journey/collection/usa.jpg",
      "type": "AB"

  "total": 20
}

说明：NT笔记，AB相册；id<0表示系统笔记相册，id>0表示用户笔记相册。暂无系统笔记

        "id": 50001,

    "hasCard": false,
    "public": false,
    "waterMark": false,
    "detail": [
        {
            "id": 1000,
            "type": "text",
            "content": "xxxxxxxx",
        },
        {
            "id": 1000,
            "type": "image",
            "url": ""
        },
        {
            "id": 1000,
            "type": "poi",
            "content": "",
            "image": "",
            "nameZh": "",
            "nameEn": "",

7. /tr-media/pc-note/update

描述：修改笔记

request:

{
    "id": 10001,
    "title":"发现西西里",
    "music": {
        "id": 50001,
        "name": "Barcelona",
        "url": "https://cdn1.zouke.com/music/Barcelona.mp3"
    },
    "hasCard": false,
    "public": false,
    "waterMark": false,
    "detail": [
        {
            "id": 1000,
            "type": "text",
            "content": "xxxxxxxx",
        },
        {
            "id": 1000,
            "type": "image",
            "url": ""
        },
        {
            "id": 1000,
            "type": "poi",
        }
    ],
    "tpl": null
}

response

{
  "code": 0,
  "id": 10001,
}

说明：若id<0，表示用户修改了系统笔记，后台会删掉该笔记并新建用户笔记，然后返回新id，前台务必用新id替换掉旧id
