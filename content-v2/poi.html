{
    "keywords": "巴黎"
}

{
    "code": 0,
    "list":[
        {
            "code": 10000,
            "name": "巴黎",

            "code": 10000,

2. /tr-media/poi/query

描述：检索POI

request：

{
    "cat": "AT",
    "keywords": "巴黎",
    "pageNo": 0,
    "pageSize": 10
}

说明：cat:POI分类，AT景点、FD美食、SH购物

response

{
  "code": 0,
  "list": [
    {
        "id": 10000,
        "image": "https://cdn1.zouke.com/poi/300002/0f0feabdf012f8796a4940f309f06490.jpg",
        "nameZh": "小象鼻山",
        "nameEn": "Falaise d'Amont",
        "score": 8,
        "star": 4,
        "rank": "埃特勒塔全部景点观光中排第2名",
        "category": "AT",
        "content": "高崖（小象鼻山）是埃特勒塔海边一串3个象鼻山中最小的一座，位于埃特勒塔镇的正北。1927年，两位飞行员试图从这里开始横跨大西洋。",
    }
  ],
  "total": 20,
  "atCnt": 3,
  "fdCnt": 4,
  "shCnt": 5
}
